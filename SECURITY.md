# Security Policy

## Supported Versions

Reviva currently supports the latest released version on `main` and the latest crates.io release.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Private Vulnerability Reporting (Security Advisories) for this repository.

Include:

- Affected version/commit
- Reproduction steps
- Expected vs actual behavior
- Impact assessment

We will acknowledge reports as quickly as possible and coordinate a fix and disclosure timeline.
