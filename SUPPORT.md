# Support

## Getting Help

For usage questions and troubleshooting:

- Open a GitHub issue with logs and reproduction steps
- Include your OS, Reviva version, backend URL, and command used

## What to Include

- Command you ran
- Error output
- Relevant `.reviva/config.toml` fields (redact secrets)
- Whether issue is reproducible

## Security Issues

For sensitive security reports, use private vulnerability reporting as described in [SECURITY.md](SECURITY.md).
